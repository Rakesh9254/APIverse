```java

.body("data.id", hasItems(7, 8))

.body("data", hasSize(2))

.body("data.email", everyItem(containsString("@")))
```

### REST Assured Implementation

```java
// Step 1: Get token
String token = given()
    .contentType(ContentType.JSON)
    .body("{ \"username\": \"user\", \"password\": \"pass\" }")
.when()
    .post("https://api.example.com/login")
.then()
    .statusCode(200)
    .extract().jsonPath().getString("token");

// Step 2: Use token
given()
    .header("Authorization", "Bearer " + token)
.when()
    .get("https://api.example.com/protected")
.then()
    .statusCode(200);
```

```java

    .body("{ \"name\": \"Test\", \"job\": \"QA\" }")

    .post("https://reqres.in/api/users")
```

2. **WHEN**: Send POST request to create user
3. **THEN**: Verify status is 201 and name matches
